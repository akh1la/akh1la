## 🚀 Featured Projects

### Multimodal AI Agent for Customer Support
> `Python` `FastAPI` `LLaMA-3` `CLIP` `Whisper` `RAG` `FAISS` `LangGraph` `Groq` `ElevenLabs`

Built a production-grade AI platform that handles customer support through
voice, text, and images — all in one pipeline.

Under the hood: a Reflective RAG system with 5,000+ knowledge chunks,
LangGraph multi-agent orchestration with intent classification,
LLM guardrails for safe outputs, and ElevenLabs TTS for voice responses.
End-to-end latency: **sub-400ms** across 50+ simulated interactions.

This wasn't a course project. It was designed, built, and optimized
the way a real production system should be.

[→ View Repository](https://github.com/akh1la/AI-Call-Agent-master)

### AI Cook – Generative AI Web App
> `Python` `Flask` `Mistral API` `Whisper` `PyTorch` `yt-dlp` `FFmpeg`

Two things in one app: drop your ingredients and get a structured,
step-by-step recipe from Mistral LLM. Or paste a YouTube cooking video
link and get the full recipe transcribed and extracted automatically
using Whisper STT — with GPU acceleration and auto English translation.

Built it because it was a fun problem. Shipped it because it actually worked.

[→ View Repository](https://github.com/akh1la/ai-cook)
